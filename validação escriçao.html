Tela de ajudar de validar inscrição

Para gerar acesso ao sistema siga os passos abaixo: usuário: Informe um nome para o usuário. Neste caso use um nome simples, ou seja, sem espaços Descrição No próximo campo informe uma descrição, a qual poderá ser um nome mais completo para o prosseguir Situação: Esta opção permite que o usuário tenha sua situação ativada no sistema. Nota: Tanto o usuário poderão ter suas permissões de uso alteradas conforme as necessidades. Caso necessite Configurações do usuário
